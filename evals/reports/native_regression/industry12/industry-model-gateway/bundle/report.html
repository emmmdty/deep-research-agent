# Model Gateway and Routing Layer

Public gateway documentation and repositories make provider routing and abstraction visible as a separate operational layer in AI infrastructure.[1][2]

The same public materials are less definitive about how much adjacent control-plane scope belongs inside that layer, so comparisons should keep the core routing claim separate from the more tentative claim about policy and observability ownership.[1][2]
