# Phase 21 — Web GUI scaffold

## Objective
Create the web GUI shell and design system foundation.

## Required outcomes
- create `apps/gui-web/`
- React + TypeScript + Vite scaffold
- shadcn/ui based component setup
- routes/layout/navigation
- API client and query/state foundation
- environment config for pointing to local API

## Acceptance
- dev app boots locally
- base layout renders
- API client config is documented
- lint/tests for GUI shell pass
- docs/gui/README.md exists

## Validation
Run at least:
- frontend install command
- frontend lint/type/test/build commands
- one local render/build smoke
